## ModalResponse Enum

### Import
```
from ArchicadDG import ModalDialog
``` 

## Example
```
from ArchicadDG import *
gt1=ModalDialog.Accept 
gt2=ModalDialog.ModalResponse.Cancel 
```

### Values
* **Accept**:The modal dialog is closed by OK button.
* **Cancel**:The modal dialog is closed by Cancel button.
