## Logger module

### Import
```
import PauperTools.Logger as Logger
``` 

### Example
[demo](../Scripts/Tests/logger_test.py)

### Functions

* **alert(msg)**
* alert message from info window
```
#msg:string
Logger.alert(msg)
```


* **write(msg)**
* print message to report window

```
#msg:string
Logger.write(msg)
```
