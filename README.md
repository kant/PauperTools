# PauperTools for Archicad

## 关于

PauperTools 是专门为Archicad建筑设计软件开发的、可让开发者使用
python开发语言对Archicad建筑设计软件进行二次开发的桥接式插件。让广大Archicad的开发者或使用者从此逃离C/C++的深渊。
